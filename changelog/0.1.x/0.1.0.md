# 0.1.0 — 2026-05-25

Initial scaffolded release. Design finalized; tools and resources defined.

## Added

- **12 tools** across two data layers (UCR Summary, NIBRS) and specialty reporting tracks: `fbi_search_agencies`, `fbi_get_agency`, `fbi_get_crime_estimates`, `fbi_get_agency_offenses`, `fbi_get_nibrs_breakdown`, `fbi_get_arrests`, `fbi_get_hate_crimes`, `fbi_get_participation`, `fbi_get_human_trafficking`, `fbi_get_leoka`, `fbi_get_arson`, `fbi_list_code_table`.
- **2 resources**: `fbi://agency/{ori}` (full agency profile), `fbi://state/{state_abbr}` (state participation overview).
- **Design doc** at `docs/design.md` — tool surface, API path mapping, domain mapping, NIBRS variable table, workflow patterns, and design decisions.
