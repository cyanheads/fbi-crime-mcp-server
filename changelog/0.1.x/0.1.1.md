# 0.1.1 — 2026-05-25

The FBI's legacy UCR backend (`crime-data-api.fr.cloud.gov`) was shut down. Field-testing found that 8 of 12 tools no longer have a live API behind them. This release reflects reality: decommissioned tools throw `ServiceUnavailable` with a recovery hint; 4 tools backed by the CDE API continue to work.

## Changed

- **`fbi_search_agencies`**, **`fbi_get_agency`**, **`fbi_get_arrests`**, **`fbi_get_hate_crimes`**, **`fbi_get_participation`**, **`fbi_get_human_trafficking`**, **`fbi_get_nibrs_breakdown`**, **`fbi_list_code_table`** — marked `[UNAVAILABLE]`; handlers throw `serviceUnavailable` on every call with a recovery hint pointing to `cde.ucr.cjis.gov`.
- **`fbi_get_crime_estimates`** — switched to CDE `/cde/summarized/` endpoint; returns monthly rates and raw counts instead of annual adjusted estimates.
- **`fbi_get_agency_offenses`** — redirected to CDE `/cde/summarized/` endpoint; provides per-agency monthly offense rates.
- **`fbi_get_arson`** — marked `[UNAVAILABLE — use fbi_get_crime_estimates]`; description directs agents to `fbi_get_crime_estimates` with `offense="arson"`.
- **`fbi_get_leoka`** — confirmed working against CDE `/cde/leoka/` endpoint; no behavior change.
- **Resources** (`fbi://agency/{ori}`, `fbi://state/{state_abbr}`) — backed by CDE summarized endpoint; return offense rates and counts rather than UCR agency profiles.
- **README** — updated to accurately list 4 active CDE tools and 8 decommissioned tools; removed misleading descriptions for decommissioned tools.
